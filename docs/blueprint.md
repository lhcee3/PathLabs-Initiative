# **App Name**: PathLabs PL Landing

## Core Features:

- Hero Section: Visually appealing hero section with headline, subtext, and background.
- Roadmap Timeline: Interactive roadmap timeline showcasing the learning progression.
- Value Proposition Cards: Display cards highlighting the unique value propositions of PATHLABS PL.
- Registration Section: Dedicated section for demo registration with a Google Form link.
- Glassmorphic Navbar: Responsive navbar with glassmorphism and a 'Join Free Demo' CTA button.
- Footer Section: Simple footer section with essential links and contact info for Telangana.

## Style Guidelines:

- Primary color: Vibrant Red (#DC2626) for a high-tech feel.
- Background color: Deep Charcoal (#0A0A0A) for a modern dark mode.
- Accent color: Slate-900 for subtle highlights and contrasts.
- Body and headline font: 'Inter' (sans-serif) for a modern and objective feel.
- Use Lucide-React icons for a consistent and clean look.
- Mobile-first, all-screen responsive layout using Tailwind CSS grid and flexbox.
- Subtle scroll reveal animations using Framer Motion for smooth transitions.
