# PATHLABS PL

Welcome to PATHLABS PL, where we transform students into tech creators. We don't just teach syntax; we build careers.

## Overview

Our roadmap takes students from foundational Python & Java all the way to professional Web Development and DSA. Join us to build a market-ready portfolio before you even finish school.

## Getting Started

To get started, take a look at `src/app/page.tsx` for the landing page structure.

## Project Structure

- `src/app/` - Next.js app directory with layout and pages
- `src/components/landing/` - Landing page components
- `src/components/ui/` - Reusable UI components
- `src/hooks/` - Custom React hooks
- `src/lib/` - Utility functions and helpers

## Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) to view the site.

## Building

```bash
npm run build
npm run start
```
